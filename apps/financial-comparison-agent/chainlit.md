# Financial Comparison Agent

Live **comparable-company analysis** for US equities — powered by **Nimble** Web Search Agents and **Claude**.

## How to use

Type a single **stock ticker symbol** in the box below, for example:

- `CMG` — Chipotle
- `CRM` — Salesforce
- `NKE` — Nike

…or click one of the **starter buttons**.

## What happens

1. **Discover peers** — the agent finds the company's public comparables.
2. **Pull financials** — current valuation multiples (P/E, EV/EBITDA, P/S, margins, growth) for the target and each peer, parsed live from the web.
3. **Find catalysts** — recent earnings, guidance, and analyst-rating moves.
4. **Verdict** — a relative-valuation read versus the peer median.

You'll watch each step run live, then get an interactive **comps table**, an **EV/EBITDA chart**, and the **verdict**. Every run is saved and can be reopened from the **Recent analyses** buttons.
