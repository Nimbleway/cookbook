# UI & UX Guidelines

## Design Philosophy

Do not build a dashboard.

Build an experience.

Think:

* Perplexity
* Linear
* Stripe
* Vercel
* Ramp

Avoid:

* Tableau
* Power BI
* Legacy analytics platforms

---

# User Attention Strategy

Conference attendees have limited attention.

Value must appear immediately.

Do not require user input before showing insights.

---

# Mobile First

Must work flawlessly via QR code.

Requirements:

* Large touch targets
* Swipeable cards
* Sticky CTA
* Fast rendering
* No horizontal scrolling

---

# CTA Design

Sticky side panel.

Mobile:

Sticky bottom CTA.

Primary CTA:

Secondary CTA:

Visit Booth
